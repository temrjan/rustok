Architecture
4 crates, clean separation
txguard
Transaction security engine. Parser, rules, simulator, enrichment. Independent crate — any wallet can use it.
qallet-core
Wallet logic. Encrypted keyring (AES-256-GCM + Argon2id), multi-chain provider, smart router, explainer.
qallet (CLI)
Command-line interface. decode, analyze, wallet new/balance/info/send. Thin wrapper over core + txguard.
qallet-api
HTTP API server (axum). POST /analyze, POST /send, GET /balance. Coming soon.
