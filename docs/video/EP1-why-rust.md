## ЗАМЕТКИ ДЛЯ СЪЁМКИ

### Подготовка перед записью
1. Открыть терминал на полный экран, шрифт крупный (16-18pt)
2. Тёмная тема терминала
3. Предварительно выполнить `cargo build` чтобы не ждать компиляцию на камеру
4. Подготовить команды в текстовом файле — копировать, не набирать руками
5. OBS: экран (display capture) + кружок (webcam, нижний правый угол)

### Кружок автора
- Размер: ~15-20% экрана
- Позиция: правый нижний угол
- Фон: нейтральный (стена, размытый)
- Одежда: без логотипов, однотонная
- Взгляд: в камеру при комментариях, на экран при демо

### Монтаж
- Между командами: jump cut (убрать время компиляции)
- JSON output: можно замедлить или подсветить ключевые поля
- Текстовые плашки: минимум. Только tagline в конце

### Музыка
- Лёгкий ambient/electronic на фоне, не мешает голосу
- Громче в hook и tagline, тише во время озвучки
